Backend API using Node Js.

How it Works:
- CRUD operation for a blog
- Messaging section to create and recieve messages from the user
- Sign in and Sign up for both the user and Admin

  
Frameworks used:
- NodeJs
- Mongo DB
- Express Js
- Passport Js with Jwt

Tools:
- Cloudinary
- circleci
- Jest
